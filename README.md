# Exercice 3 : Programme de Comptage de Mots - Java

## Description
Ce projet est un utilitaire Java permettant de compter le nombre total de mots contenus dans un fichier texte volumineux. Il a été réalisé dans le cadre du cours de DevOps à Pigier Côte d'Ivoire.

## Fonctionnement
- Lecture d'un fichier externe via la classe `Scanner`.
- Analyse ligne par ligne du contenu.
- Découpage des chaînes de caractères en utilisant les espaces comme séparateurs.
- Calcul et affichage de la somme globale des mots.

## Instructions
1. Placez le fichier texte à analyser à la racine du projet.
2. Modifiez le nom du fichier dans le code si nécessaire.
3. Exécutez la classe `Main`.
